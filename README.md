# Plank Sack Tracker
A RuneLite plugin designed to show a highly customizable overlay of the contents of the plank sack and show an overlay and/or tooltip of the current contents.

The plugin supports tracking individual plank types in the sack. It has multiple overlay style options and is designed to support the new planks from sailing.

It allows for customizing the display of icons, the overlay and tooltip font colors, the presence of labels, and the arrangement of the overlay.
